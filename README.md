## ✨ About

This is my personal take on the original shell by [ilyamiro](https://github.com/ilyamiro) — huge thanks to them for the original design and work, all credit goes their way. I just liked this style enough that I wanted to keep developing it my own way and share it with everyone.

It can now be installed using the included installer, so setup should be quick and easy.

More updates and improvements are on the way, so stay tuned.
