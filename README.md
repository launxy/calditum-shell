## ✨ About

This is my personal take on the original shell by [ilyamiro](https://github.com/ilyamiro) — huge thanks to them for the original design and work, all credit goes their way. I just liked this style enough that I wanted to keep developing it my own way and share it with everyone.

It can now be installed using the included installer, so setup should be quick and easy.

More updates and improvements are on the way, so stay tuned.

## 📦 Installation

Just run the installer and you should be good to go.

---

## 🔧 Changes from the original

- Matugen now generates monochrome colors for black and white wallpapers.
- The `movies` and `stewart` modules have been disabled.
- The topbar has been simplified, removing the color gradients.
- The topbar now includes a new module to open `wvkbd` (virtual keyboard).

---

## 🐛 Known bugs

- The weather module is not properly aligned (at least on my machine).
